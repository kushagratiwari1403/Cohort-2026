# Cursor Website Clone Assignment

I have recreated the main landing page of Cursor.com using HTML and CSS. This assignment focuses on pixel-perfect design, grid layouts, and modern dark-themed UI.

## Sections Recreated
I have built the entire homepage, including:

* **Navigation Bar:** Sticky top bar with logo, links, and action buttons.
* **Hero Section:** Main title, download CTA, and hero images.
* **Feature Blocks:** Detailed sections for "Agent", "Tab Autocomplete", and "Codebase Understanding" with alternating layouts.
* **Social Proof:**
    * **Trusted Logos:** Grid of companies like OpenAI, Stripe, etc.
    * **Testimonials:** Review cards from industry leaders (Andrej Karpathy, shadcn, etc.).
* **Changelog & Highlights:** Updates section showing recent versions and blog posts.
* **Footer:** Complete footer with links, copyright, and theme/language controls.

---

## Design Details

### Fonts
Used a custom font setup to match the original:
* CursorGothic (Custom font family)
* Fallbacks: sans-serif

### Colors Used
* **Backgrounds:** #14120b (Main), #1b1913
* **Primary Text:** #edecec
* **Accents:** #f54e00
* **Borders:** #3f3f46

---

## Technology Used
* **HTML5:** Semantic structure (nav, main, section, footer).
* **CSS3:**
    * **Flexbox:** For alignment in nav, buttons, and rows.
    * **CSS Grid:** Used for the Reviews and Links sections.
    * **Variables:** For consistent colors.
* **Vercel:** For hosting.
